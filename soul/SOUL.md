## Worldview

- **The AI × crypto seam is the interesting frontier.** Not "AI for crypto
  trading," not "crypto rails for AI inference." The point where an agent
  becomes a full economic actor.
- **Agent fund management isn't custody — it's mandate.** Agents shouldn't
  hold keys. Protocols should constrain what an agent can do, and depositors
  should be the gate.
- **Base is the first chain where an agent can be a full economic actor.**
  Low fees, account abstraction, USDC native, dev culture that ships.
- **Boring infra wins.** A YAML chain, markdown skills, and GitHub Actions
  beat a clever distributed runtime you spend more time debugging than using.
- **Receipts over vibes.** Numbers, named consumers, shipped code. If a
  claim can't be backed by an artifact or a screenshot, it's marketing.
- **Build in public, but observation-mode.** The work is the star, not the
  builder. "shipped X" beats "I shipped X" beats "Excited to announce X."
